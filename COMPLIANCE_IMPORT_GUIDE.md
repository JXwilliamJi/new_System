# 合规管理模块 - Excel导入功能使用指南

## 功能概述

合规管理模块新增了一键导入功能，支持从Excel文件中直接导入合规检查数据。

## 功能特性

### 1. 智能识别Sheet页
- 自动查找包含"合规审核"字样的Sheet页
- 如果找不到对应Sheet页，会提示用户

### 2. 智能数据扫描
- 自动识别表头行（包含"类型"、"标题"、"检查"等关键字）
- 从有类型数据的行开始扫描
- 自动跳过空行和不相关的行

### 3. 数据验证
- 必填字段验证（标题、类型）
- 用户信息自动匹配（经办人、审核人、复核人）
- 文件格式和大小验证

### 4. 导入反馈
- 实时进度显示
- 详细的导入结果统计
- 错误详情展示

## Excel文件格式要求

### Sheet页要求
- 必须包含"合规审核"字样的Sheet页名称
- 例如："合规审核数据"、"合规审核表"、"合规审核"等

### 数据列顺序
建议按照以下顺序排列列：

| 列号 | 列名 | 是否必填 | 说明 |
|------|------|----------|------|
| 1 | 标题 | ✅ 是 | 合规检查标题 |
| 2 | 类型 | ✅ 是 | 合规检查类型（如：数据安全、反洗钱、反腐败等） |
| 3 | 提交部门 | 否 | 提交部门名称 |
| 4 | 经办人 | 否 | 经办人姓名（需与系统用户姓名一致） |
| 5 | 检查日期 | 否 | 日期格式：YYYY-MM-DD |
| 6 | 检查结果 | 否 | 检查结果描述 |
| 7 | 状态 | 否 | pending（待检查）、completed（已完成）、failed（未通过） |
| 8 | 审核人 | 否 | 审核人姓名（需与系统用户姓名一致） |
| 9 | 复核人 | 否 | 复核人姓名（需与系统用户姓名一致） |
| 10 | 描述 | 否 | 检查描述 |

## 使用步骤

### 1. 准备Excel文件
按照上述格式要求准备Excel文件，确保：
- 包含"合规审核"字样的Sheet页
- 第一行为表头行
- 数据从第二行开始
- 标题和类型列有数据

### 2. 登录系统
使用管理员或普通用户账号登录系统

### 3. 进入合规管理模块
点击左侧菜单栏的"合规管理"

### 4. 点击导入按钮
在页面右上角点击"导入Excel"按钮

### 5. 选择文件
在弹出的模态框中点击"选择文件"，选择准备好的Excel文件

### 6. 开始导入
点击"开始导入"按钮，系统会：
- 验证文件格式
- 解析Excel数据
- 导入到数据库
- 显示导入结果

### 7. 查看结果
导入完成后会显示：
- 成功导入数量
- 失败数量
- 总计数量
- 错误详情（如有）

## 注意事项

### 1. 文件格式
- 仅支持 .xlsx 和 .xls 格式
- 文件大小不能超过 10MB

### 2. 数据要求
- 标题和类型为必填项
- 用户姓名需与系统中已存在的用户姓名一致
- 日期格式需为 YYYY-MM-DD

### 3. Sheet页名称
- 必须包含"合规审核"字样
- 系统会自动查找第一个包含该字样的Sheet页

### 4. 数据扫描
- 系统会自动识别表头行
- 从有类型数据的行开始扫描
- 自动跳过空行

## 常见问题

### Q1: 提示"未找到包含'合规审核'字样的sheet页"
**A:** 请检查Excel文件中的Sheet页名称是否包含"合规审核"字样。

### Q2: 提示"标题和类型不能为空"
**A:** 请检查Excel文件中是否有空的标题或类型列。

### Q3: 经办人、审核人、复核人无法匹配
**A:** 请确保Excel中的姓名与系统中已存在的用户姓名完全一致。

### Q4: 导入失败
**A:** 请检查：
1. 文件格式是否正确（.xlsx 或 .xls）
2. 文件大小是否超过 10MB
3. Excel格式是否符合要求

### Q5: 部分数据导入失败
**A:** 系统会显示详细的错误信息，可以根据错误提示修改数据后重新导入。
